# FCKT 2025 시즌 결산

2025 시즌 FCKT 축구팀 결산 웹사이트

## 🚀 Vercel 배포 방법

### 방법 1: 웹사이트에서 직접 배포 (가장 쉬움!)

1. **Vercel 가입**
   - [vercel.com](https://vercel.com) 접속
   - "Sign Up" 클릭
   - GitHub, GitLab, 또는 이메일로 가입

2. **프로젝트 배포**
   - 로그인 후 "Add New..." → "Project" 클릭
   - "Import Git Repository" 또는 "Deploy from template" 선택
   - 이 프로젝트 폴더를 ZIP으로 압축해서 업로드하거나
   - GitHub에 올린 후 연결

3. **자동 설정**
   - Vercel이 자동으로 Vite 프로젝트 감지
   - "Deploy" 버튼 클릭

4. **완료!**
   - 2-3분 후 `https://your-project.vercel.app` 주소 생성
   - 이 주소를 팀원들에게 공유하세요! 🎉

### 방법 2: Vercel CLI 사용 (터미널 사용 가능한 경우)

```bash
# Vercel CLI 설치
npm i -g vercel

# 프로젝트 폴더에서 실행
vercel

# 질문에 답변 (대부분 엔터키만 누르면 됨)
```
